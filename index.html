PROJECT
By Demi, Tofe, Ryanne and Janine

During the Digital Media Sprintweek, we tried create a decentralized network. We thought of making a "program", in which you can share text based files, that works via a Partial Mesh Network.

You can see all the steps we took by exploring this website (which shows the prototype itself), or you can read more about it by clicking on the link below.
